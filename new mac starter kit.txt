1. Install command-line tools for XCode

$ xcode-select --install

2. Install HomeBrew

$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

3. Install Cask

$ brew install cask

4. Install HomeBrew applications

$ brew install git
$ brew install git-flow
$ brew install hub
$ brew install irssi
$ brew install node
$ brew install p7zip
$ brew install python3
$ brew install telnet
$ brew install tmux
$ brew install todo-txt
$ brew install vim --with-override-system-vi --with-python3

6. Clone my dotfiles repo

$ git clone https://github.com/justinmpeterson/documentation.git

7. Download Vundle for plugins for vim

$ git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim

8. Do not create .DS_Store files on network shares

$ defaults write com.apple.desktopservices DSDontWriteNetworkStores true

9. Save screenshots somewhere else

$ mkdir ~/Desktop/Screenshots; defaults write com.apple.screencapture location ~/Desktop/Screenshots; killall SystemUIServer
