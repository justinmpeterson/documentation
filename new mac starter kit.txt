1. Install command-line tools for XCode

$ xcode-select --install

2. Install HomeBrew

$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

3. Install Cask

$ brew install cask

4. (Optional) Grant extra permissions on the Homebrew directory

$ sudo mkdir $(brew --prefix)/include
$ sudo mkdir $(brew --prefix)/Frameworks
$ sudo chown -R $(whoami) $(brew --prefix)/*

7. Clone my dotfiles repo

$ mkdir -p ~/Documents/code/documentation; git clone https://github.com/justinmpeterson/documentation.git ~/Documents/code/documentation

8. Download Vundle for plugins for vim

$ git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim

9. Do not create .DS_Store files on network shares

$ defaults write com.apple.desktopservices DSDontWriteNetworkStores true

10. Save screenshots somewhere else

$ mkdir ~/Desktop/Screenshots; defaults write com.apple.screencapture location ~/Desktop/Screenshots; killall SystemUIServer

11. Deploy some symbolic links

$ ln -s $HOME/Documents/code/documentation/dotfiles/.bash_profile $HOME/.bash_profile
$ ln -s $HOME/Documents/code/documentation/dotfiles/.todo.cfg $HOME/.todo.cfg
$ ln -s $HOME/Documents/code/documentation/dotfiles/.tmux.conf $HOME/.tmux.conf
$ ln -s $HOME/Documents/code/documentation/dotfiles/.vimrc $HOME/.vimrc
